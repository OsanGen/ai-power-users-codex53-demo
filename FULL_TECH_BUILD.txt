## 2) What this repo is
WONDERLAND is a browser-first vanilla JS canvas game that teaches neural-net concepts through floor-based progression, lesson overlays, and upgrade loops. It ships as a static web app with deterministic QA hooks (`window.advanceTime`, `window.render_game_to_text`) and script-driven smoke checks.

## 3) System map
- Components: `index.html`, `src/main.js`, `src/core.js`, `src/systems.js`, `src/render.js`, `src/lesson-diagram-v2.js`, `src/audio.js`, `src/share.js`, `src/content.js`, `src/upgrades.js`, `src/ui-tokens.js`, `narrative.js`.
- Boundaries: `main.js` owns loop/bootstrap; `systems.js` owns simulation/state/input; `render.js` owns canvas draw pipeline and overlays; `lesson-diagram-v2.js` owns semantic lesson-diagram rendering with Pixi/ELK optional acceleration and hard fallback behavior.
- Key data flow: narrative/card `visualMode` -> lesson overlay renderer -> `drawLessonSlideDiagram` -> `AIPU.lessonDiagramV2.renderTo` (or fallback sketch) -> debug snapshot in `render_game_to_text`.
- If DEV MODE is active, `final_build/BUILD_MAP.txt` is the canonical system map.

## 4) Entrypoints and run commands
- Local run: `python3 -m http.server 4173` then open `http://127.0.0.1:4173/index.html`.
- Build: static browser game (no transpile/bundle step).
