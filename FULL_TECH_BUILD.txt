## 2) What this repo is
WONDERLAND is a browser-first vanilla JS canvas game that teaches neural-net concepts through floor-based progression, lesson overlays, and upgrade loops. It ships as a static web app with deterministic QA hooks (`window.advanceTime`, `window.render_game_to_text`) and script-driven smoke checks.

## 3) System map
- Components: `index.html`, `src/main.js`, `src/core.js`, `src/systems.js`, `src/render.js`, `src/lesson-diagram-v2.js`, `src/audio.js`, `src/share.js`, `src/content.js`, `src/upgrades.js`, `src/ui-tokens.js`, `narrative.js`.
- Boundaries: `main.js` owns loop/bootstrap; `systems.js` owns simulation/input; `render.js` owns canvas draw pipeline and overlays; `lesson-diagram-v2.js` owns semantic lesson-diagram rendering with optional Pixi/ELK acceleration and deterministic fallbacks.
- Key data flows and boundaries: narrative `visualMode` selects mode spec -> `render.js` `drawLessonSlideDiagram` -> `AIPU.lessonDiagramV2.renderTo` -> canvas draw path and debug payload in `render_game_to_text`.
- If DEV MODE is active, reference `final_build/BUILD_MAP.txt` as detailed architecture map.
